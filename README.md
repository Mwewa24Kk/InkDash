# InkDash Print Web App

A static multi-page website for InkDash Print, an online printing and delivery service in Lusaka. The site highlights services, pricing ideas, and provides pages to place orders, sign up, and get in touch.

## Pages
- `index.html`: Landing page with service overview and calls to action.
- `makeorder.html`: Order form for print requests.
- `contact.html`: Contact details and inquiry form.
- `login.html`: Customer login page.
- `signup.html`: Account creation page.

## Assets
- `theme.css`: Global styles and layout system.
- `app.js`: Front-end scripts (if used).
- `Icons/`: Icon assets.

## Run Locally
1. Open any page directly in your browser (for example, `index.html`).
2. Or serve the folder with a simple static server for relative links:

```bash
# Python 3
python -m http.server 8000
```

Then visit `http://localhost:8000/index.html`.

## Customize
- Update colors, typography, and layout in `theme.css`.
- Edit page content directly in the HTML files.
